<Section icon={Shield} title="Troubleshooting Your Kill Sound">

If your **legends battlegrounds kill sound id** is not playing, there are several common causes. Most issues are related to Roblox's platform-wide audio settings rather than the game itself.

| Common Issue | Likely Cause | Solution |
| :--- | :--- | :--- |
| **No Sound on Kill** | Audio is private/deleted | Replace with a Public ID |
| **Sound is too quiet** | Low original upload volume | Find a "Loud" or "Boosted" version |
| **Sound plays late** | Server latency (Ping) | Check your connection to the Roblox server |
| **Wrong sound plays** | Typo in the ID box | Re-copy the ID from the URL carefully |
| **Sound cuts off** | Game engine priority | Ensure the sound file is short (under 5s) |

<Callout type="warning" title="Volume Warning">
Avoid using "Earrape" or excessively loud IDs. These can be reported by other players and may lead to a temporary mute or ban from the game's social features, as it disrupts the experience for others.
</Callout>
