<Section icon={Sparkles} title="Mecánicas de Shinji y Virus Overlord">

La siguiente fase de actualizaciones se centra principalmente en refinar a Shinji e introducir a Virus Overlord al público en general. Shinji está programado para recibir sus remates y un cuarto movimiento definitivo, lo que lo convierte en un combatiente muy versátil. Sin embargo, el desarrollador Linste confirmó que los remates de Shinji contarán con dos cinemáticas en lugar de las tres planeadas originalmente, asegurando una mayor calidad visual para las animaciones restantes.

| Personaje | Actualizaciones planeadas | Mecánicas clave | Tier esperado |
| | | | |
| **Shinji** | 4ta Definitiva, Remates | Combos de alta velocidad, remates cinemáticos | **Alto** |
| **Virus Overlord** | Nueva mecánica | Utilidad poderosa, definitivas duales | **Superior** |

Se espera que Virus Overlord sacuda el meta competitivo. Los desarrolladores han adelantado una mecánica completamente nueva diseñada para hacer que el personaje sea muy formidable en los intercambios neutrales. Si la comunidad selecciona la Opción 1, Virus Overlord se lanzará en acceso anticipado con un conjunto de movimientos base completo y dos movimientos definitivos distintos.
