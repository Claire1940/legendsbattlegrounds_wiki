<Section icon={Settings} title="Comandos e Configurações de Administrador Essenciais">

A interface administrativa em *Legends Battlegrounds* permite um controle detalhado sobre o ciclo de combate. A maioria dos **comandos do servidor privado de legends battlegrounds** é acessada através de um painel de interface dedicado, mas entender a lógica por trás dessas configurações é fundamental para criar um ambiente de treino produtivo.

### Manipulação da Lógica de Combate
Ao substituir os scripts de combate padrão, você pode testar interações específicas. Por exemplo, desativar os tempos de recarga permite que você pratique o combo de acompanhamento "Tiger Palm" do **Martial Legend** repetidamente até que o tempo se torne memória muscular.

| Comando/Configuração | Ação Realizada | Melhor Caso de Uso |
| :--- | :--- | :--- |
| **Desativar Tempos de Recarga** | Remove o tempo de espera de todas as habilidades | Praticar conexões de movimentos específicas |
| **Conceder Despertar** | Preenche instantaneamente a barra G | Testar **Warrior's Grief** ou **No More Games** |
| **Forçar Finalizador** | Ativa a cena cinematográfica em qualquer eliminação | Criação de conteúdo e demonstração de movimentos |
| **Ativar Movimentos Duplos** | Permite misturar os kits de dois personagens | Builds experimentais e partidas divertidas |
| **Redefinir Tudo** | Restaura a vida e os tempos de recarga | Iniciar uma nova rodada de duelo |
| **Expulsar Jogador** | Remove um usuário específico | Moderar torneios privados |

<Callout type="warning" title="Importante">
Ativar **Dual Movesets** pode causar bugs visuais se duas animações conflitantes forem ativadas simultaneamente. Use este recurso principalmente para diversão experimental, em vez de testes de equilíbrio sérios.
</Callout>

### Controles de Ambiente e Interface
Além do combate, os proprietários do servidor podem ajustar os aspectos visuais e técnicos da partida. Isso inclui alternar o "Modo Fotossensível" para todos os participantes ou ajustar o "Deslocamento do Shift-lock" para garantir que todos joguem em igualdade de condições durante eventos organizados.

<Section icon={Swords} title="Guia de Configuração do Modo de Treino">

Configurar um ambiente de treinamento profissional é o principal motivo pelo qual os jogadores buscam os **comandos do servidor privado de legends battlegrounds**. Para dominar personagens como **Jun** ou o de acesso antecipado **Shinji**, você precisa de uma configuração que permita falhas repetitivas e redefinições instantâneas.

<Steps>
  <Step num="1" title="Inicializar a Arena de Prática">
    Entre no seu servidor privado e abra as Configurações de Administrador. Ative **Desativar Tempos de Recarga** e **Vida Infinita** (se disponível) para seu parceiro de treino ou conta de teste (dummy).
  </Step>
  <Step num="2" title="Configurar as Definições de Ragdoll">
    Em *Legends Battlegrounds*, a recuperação de ragdoll é uma variável importante. Use comandos para alternar a **Evasiva Automática** (Auto-Evasive) no seu alvo para ver quais dos seus combos são "verdadeiros" (true combos) e quais podem ser evitados com um Q-dash.
  </Step>
  <Step num="3" title="Dominar o Combo de Parede (Wall Carry)">
    Posicione o seu alvo perto de uma parede. Pratique a **Sequência de Parede do Jun** ou o **Combo de Parede do Martial Legend**. Use o comando de redefinir frequentemente para se manter na posição ideal sem ter que perseguir o alvo pelo mapa.
  </Step>
  <Step num="4" title="Prática de Transição para o Despertar">
    Conceda a si mesmo o despertar usando o comando do servidor. Pratique a transição de um combo do kit básico para a ativação do despertar, como o **Dragon's Fury do Martial Legend**, para garantir que você não esteja deixando uma brecha para o oponente contra-atacar.
  </Step>
</Steps>

<Callout type="success" title="Domínio de Combos">
Usar a configuração **No Cooldown** é a maneira mais rápida de aprender a técnica "Wolf Fang Miss Tech" do Jun. Você pode tentar a redefinição de flip-dash dezenas de vezes por minuto, em comparação com uma vez a cada poucos segundos em uma partida pública.
</Callout>

</Section>
