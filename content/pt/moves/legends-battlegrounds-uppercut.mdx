A eficácia do uppercut depende muito da sua sequência anterior de M1. Para a maioria das lendas, o padrão "3M1 + Uppercut" é o básico. No entanto, com a evolução do combate em 2026, os melhores jogadores agora integram buffers de pulo (jump-buffering) e esquivas laterais (side-dashing) para manipular a física de ragdoll do oponente, tornando os golpes seguintes quase impossíveis de esquivar.

| Mecânica | Descrição | Benefício Principal |
| :--- | :--- | :--- |
| **Lançador Padrão** | 3M1 seguido por um Uppercut neutro | Extensão básica de combo para iniciantes |
| **Uppercut com Pulo** | Pular simultaneamente com o comando do Uppercut | Aumenta o tempo no ar para movimentos subsequentes |
| **Uppercut Rápido** | Cancelar a recuperação do 3º M1 em Uppercut | Cria um "True Combo" que é inescapável |
| **Uppercut Atrasado** | Esperar o oponente começar a cair | Atrai o uso precoce de Evasiva do inimigo |

<Callout type="tip" title="Dica de Tempo Profissional">

<Section icon={Sparkles} title="Sinergia de Uppercut e Fang Dance de Jun">

O estilo de jogo de Jun no *Legends Battlegrounds* é significativamente mais fluido do que o de Martial Legend. Seu uppercut é usado principalmente para preparar o "Fang Dance" ou o "Glimpse of Power". No circuito competitivo de 2026, os jogadores de Jun usam o uppercut para criar variantes de "Virgule Cascade" que arremessam o oponente no ar, permitindo um loop secundário de M1.

**Passos para Dominar o Combo One-Shot de Jun:**

<Steps>
  <Step num="1" title="A Preparação">
    Comece com uma Esquiva para Frente seguida de 3 M1s. É cruel acertar todos os três golpes para acumular hit-stun suficiente. Siga com um Downslam para testar se o oponente tem uma Evasiva pronta.
  </Step>
  <Step num="2" title="A Técnica Tearing Fang">
    Após o Downslam, espere o máximo de tempo possível para reduzir seus tempos de recarga e, em seguida, inicie a Técnica Tearing Fang. Isso puxa o oponente de volta para o seu alcance de combate corpo a corpo.
  </Step>
  <Step num="3" title="A Transição do Uppercut">
    Execute uma Esquiva Lateral seguida de uma Esquiva para Frente para fechar a distância. Acerte mais 3 M1s e ative imediatamente o **uppercut no Legends Battlegrounds**. Esta é a janela mais crítica.
  </Step>
  <Step num="4" title="O Finalizador Virgule">

<Section icon={Shield} title="Contra-ataque Defensivo e Tipos de Combo">

Entender o **uppercut no Legends Battlegrounds** também significa saber como se defender dele. O combate neste jogo é categorizado em três tipos distintos de combo. Reconhecer qual deles seu oponente está tentando ditará sua estratégia defensiva.

1.  **True Combos**: São sequências inescapáveis onde a duração do hit-stun excede o tempo de início do próximo movimento. Se você for atingido por um uppercut em um True Combo, sua única opção é usar uma Evasiva.
2.  **Combos Regulares**: Estes possuem pequenas brechas. Se um oponente errar o tempo do seu **uppercut no Legends Battlegrounds**, você pode usar um "Cancelamento de Esquiva" (Dash Cancel) para escapar da sequência sem gastar sua barra de Evasiva.
3.  **Combos One-Shot**: São sequências especializadas projetadas para levar um jogador de 100% a 0% de vida. Eles quase sempre utilizam um uppercut com buffer de pulo para maximizar o tempo no ar.

| Tipo de Combo | Característica Chave | Melhor Defesa |
| :--- | :--- | :--- |
| **True Combo** | Sem brechas entre os golpes | Evasiva (Barra Azul) |
| **Combo Regular** | Escapável através do tempo correto | Cancelamento de Esquiva / Esquiva Lateral |
| **One Shot** | Mata com a vida cheia | Não seja atingido pela abertura |
| **Infinito** | Usa a técnica "Not Done Yet" | Tempo perfeito de Evasiva |

<Callout type="info" title="Mecânica de Combate">
Um "Cancelamento de Esquiva" (Dash Cancel) é executado inserindo um comando de esquiva durante o primeiríssimo frame de uma brecha de combo que não seja true. Esta é a marca registrada de um jogador de alta habilidade no *Legends Battlegrounds*.
</Callout>
