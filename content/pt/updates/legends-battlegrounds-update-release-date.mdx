<Section icon={Sparkles} title="Mecânicas Avançadas de Combate e Mudanças Apelonas (OP)">

A parte mais emocionante da próxima atualização não são apenas os personagens, mas as mudanças fundamentais na lógica de combate. O Virus Overlord, em particular, está programado para introduzir uma mecânica que nunca foi vista no gênero de battlegrounds no Roblox.

Embora os detalhes exatos estejam sob NDA (Acordo de Confidencialidade), os testadores deram a entender que isso envolve uma nova maneira de interagir com o ambiente ou com o tempo de recarga (cooldown) dos inimigos. Isso torna o Virus Overlord um item "obrigatório" para os jogadores que gostam de superioridade técnica em combate.

| Categoria da Mecânica | Descrição | Nível de Impacto |
| :--- | :--- | :--- |
| Finalizações em Cutscene | Animações de alta fidelidade para Shinji | Alto |
| Mecânica Única OP | Sistema secreto do Virus Overlord | Extremo |
| Variantes de Ultimates | Múltiplas maneiras de encerrar uma luta | Médio |
| Quadros de Impacto | Retorno visual para golpes pesados | Estético |

<Callout type="info" title="Nota Importante">
A finalização de "Quadro de Impacto" do Shinji foi recentemente substituída por uma cutscene mais dinâmica porque a original parecia "ruim" para os desenvolvedores. Isso garante que, embora haja menos finalizações, as que restaram são de altíssimo nível.
</Callout>
