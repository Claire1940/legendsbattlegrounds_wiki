<Section icon={Settings} title="Основные админ-команды и настройки">

Административный интерфейс в *Legends Battlegrounds* позволяет осуществлять детальный контроль над боевым процессом. Большинство **команд приватного сервера Legends Battlegrounds** доступны через специальную панель интерфейса, но понимание логики этих настроек является ключом к созданию продуктивной тренировочной среды.

### Манипуляция боевой логикой
Переопределяя стандартные боевые скрипты, вы можете тестировать специфические взаимодействия. Например, отключение перезарядки позволяет многократно практиковать продолжение «Tiger Palm» у **Martial Legend**, пока тайминг не закрепится в мышечной памяти.

| Команда/Настройка | Выполняемое действие | Лучший случай использования |
| :--- | :--- | :--- |
| **Disable Cooldowns** | Убирает время ожидания навыков | Отработка связок конкретных приемов |
| **Grant Awakening** | Мгновенно заполняет G-шкалу | Тестирование **Warrior's Grief** или **No More Games** |
| **Force Finisher** | Активирует добивание при любом убийстве | Создание контента и демонстрация приемов |
| **Enable Dual Moveset** | Позволяет смешивать наборы двух героев | Экспериментальные билды и фановые матчи |
| **Reset All** | Восстанавливает здоровье и откаты | Начало нового раунда дуэли |
| **Kick Player** | Удаляет конкретного пользователя | Модерация приватных турниров |

<Callout type="warning" title="Важно">
Включение **Dual Movesets** может вызвать визуальные баги, если одновременно сработают две конфликтующие анимации. Используйте эту функцию в первую очередь для экспериментов и веселья, а не для серьезного тестирования баланса.
</Callout>

### Управление окружением и интерфейсом
Помимо боя, владельцы серверов могут настраивать визуальные и технические аспекты матча. Это включает переключение «Фоточувствительного режима» для всех участников или настройку «Смещения Shift-lock», чтобы гарантировать равные условия игры для всех во время организованных мероприятий.

<Section icon={Swords} title="Руководство по настройке режима тренировки">

Создание профессиональной тренировочной среды — основная причина, по которой игроки ищут **команды приватного сервера Legends Battlegrounds**. Чтобы освоить таких персонажей, как **Jun** или доступный по раннему доступу **Shinji**, вам нужна настройка, позволяющая совершать многократные ошибки и мгновенно сбрасывать состояние.

<Steps>
  <Step num="1" title="Инициализация тренировочной арены">
    Зайдите на свой приватный сервер и откройте настройки администратора (Admin Settings). Включите **Disable Cooldowns** и **Infinite Health** (если доступно) для вашего партнера по тренировке или твинк-аккаунта.
  </Step>
  <Step num="2" title="Настройка параметров Ragdoll">
    В *Legends Battlegrounds* восстановление из состояния ragdoll — важная переменная. Используйте команды, чтобы включить **Auto-Evasive** на вашей цели, чтобы увидеть, какие из ваших комбо являются «истинными» (true), а из каких можно сбежать с помощью Q-рывка.
  </Step>
  <Step num="3" title="Освоение комбо у стены (Wall Carry)">
    Расположите цель рядом со стеной. Практикуйте **Jun Wall Sequence** или **Martial Legend Wall Combo**. Часто используйте команду сброса, чтобы оставаться в оптимальной позиции, не преследуя цель по всей карте.
  </Step>
  <Step num="4" title="Практика перехода в пробуждение">
    Даруйте себе пробуждение с помощью команды сервера. Практикуйте переход из базового комбо в активацию пробуждения, например, **Dragon's Fury** у Martial Legend, чтобы убедиться, что вы не оставляете окна для контратаки противника.
  </Step>
</Steps>

<Callout type="success" title="Мастерство комбо">
Использование настройки **No Cooldown** — самый быстрый способ выучить «Wolf Fang Miss Tech» для Jun. Вы можете пробовать сброс флип-рывка десятки раз в минуту по сравнению с одним разом в несколько секунд в публичном матче.
</Callout>

</Section>

### Балансировка персонажей в приватных матчах
Поскольку владелец сервера может видеть данные конкретных игроков, вы можете проводить матчи на основе уровней силы (Tier-based). Например, вы можете устроить турнир «Только Martial Legend», чтобы сосредоточиться на чистой игре в нейтрале и таймингах контратак.

| Персонаж | Роль в турнире | Фокус команд |
| :--- | :--- | :--- |
| **Martial Legend** | Сбалансированный нейтрал | Фокус на тайминге **Triple Kick Counter** |
| **Jun** | Техничный специалист | Фокус на вариантах **Virgule Cascade** |
| **Shinji** | Агрессивный прессинг | Фокус на циклах сброса **Flashstep Fury** |

<Callout type="info" title="Стратегия эксперта">

Используя свой приватный сервер, вы гарантируете место для себя и своих друзей, чтобы изучить пробуждение Shinji **Warrior's Grief** без лагов и помех.
