Эффективность апперкота сильно зависит от предшествующей последовательности M1. Для большинства легенд стандарт «3M1 + апперкот» является базовым. Однако, по мере развития боевой системы в 2026 году, топовые игроки теперь интегрируют буферизацию прыжка и боковые рывки (side-dash), чтобы манипулировать физикой «тряпичной куклы» противника, делая последующие удары практически невозможными для уклонения.

| Механика | Описание | Основное преимущество |
| :--- | :--- | :--- |
| **Стандартный запуск** | 3M1 с последующим нейтральным апперкотом | Базовое продолжение комбо для новичков |
| **Апперкот в прыжке** | Прыжок одновременно с вводом апперкота | Увеличивает время нахождения в воздухе для последующих приемов |
| **Быстрый апперкот** | Отмена восстановления 3-го M1 в апперкот | Создает неизбежное «Настоящее комбо» |
| **Задержанный апперкот** | Ожидание момента, когда противник начнет падать | Выманивает раннее использование Evasive у врага |

<Section icon={Shield} title="Защитная контратака и типы комбо">

Понимание **апперкота в Legends Battlegrounds** также означает знание того, как от него защищаться. Бой в этой игре делится на три различных типа комбо. Распознавание того, какое из них пытается провести ваш противник, продиктует вашу защитную стратегию.

1.  **Настоящие комбо (True Combos)**: Это неизбежные последовательности, где длительность хит-стана превышает время запуска следующего приема. Если вы попали под апперкот в Настоящем комбо, ваш единственный вариант — использовать Evasive.
2.  **Обычные комбо**: В них есть небольшие промежутки. Если противник ошибся с таймингом своего **апперкота в Legends Battlegrounds**, вы можете использовать «Dash Cancel», чтобы выйти из последовательности, не тратя шкалу Evasive.
3.  **Ваншот-комбо**: Это специализированные цепочки, предназначенные для доведения игрока со 100% до 0% ОЗ. Они почти всегда используют апперкот с буферизацией прыжка для максимизации времени в воздухе.

| Тип комбо | Ключевая характеристика | Лучшая защита |
| :--- | :--- | :--- |
| **Настоящее комбо** | Нет промежутков между ударами | Evasive (синяя шкала) |
| **Обычное комбо** | Можно избежать при правильном тайминге | Dash Cancel / Side Dash |
| **Ваншот** | Убивает при полном ОЗ | Не попадаться на открывающий удар |
| **Бесконечное** | Использует технику «Not Done Yet» | Идеальный тайминг Evasive |

<Callout type="info" title="Боевая механика">
«Dash Cancel» выполняется путем ввода команды рывка в самый первый кадр промежутка в ненастоящем комбо. Это признак высококвалифицированного игрока в *Legends Battlegrounds*.
</Callout>
