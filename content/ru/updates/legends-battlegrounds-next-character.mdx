- **Решения по опросам**: Разработчики предложили два разных варианта, балансирующих между скоростью выпуска и проработкой персонажей.

Следующий этап обновлений в основном сосредоточен на доработке Синдзи и представлении Virus Overlord широкой публике. Синдзи должен получить свои добивания и четвертый ультимативный прием, что сделает его очень универсальным бойцом. Однако разработчик Linste подтвердил, что добивания Синдзи будут включать две кат-сцены вместо изначально запланированных трех, чтобы обеспечить более высокое визуальное качество оставшихся анимаций.

| Персонаж | Планируемые обновления | Ключевые механики | Ожидаемый тир |
| | | | |
| **Синдзи** | 4-й ультимейт, добивания | Высокоскоростные комбо, кат-сцены добиваний | **Высокий** |
| **Virus Overlord** | Совершенно новая механика | Мощные вспомогательные приемы, двойные ультимейты | **Лучший** |

Ожидается, что Virus Overlord встряхнет соревновательную мету. Разработчики протизерили совершенно новую механику, разработанную для того, чтобы сделать персонажа грозным соперником в нейтральной игре. Если сообщество выберет Вариант 1, Virus Overlord выйдет в раннем доступе с полным базовым набором приемов и двумя различными ультимативными способностями.
